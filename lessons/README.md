# 🌲 Cypress, from Zero to the Cloud 

Sample project for the "Cypress, from Zero to the Clound" course of the talking about testing online school.

## Pre-requirements

It is required to have git, Node.jse and npm installed to clone and run this project.

>I've used versions `2.42.1`, `v20.13.1` and `10.8.1` of git, Node.js and npm, respectively. I suggest you use the same or later LTS versions.

### Installation

Run `npm install`  (or `npm i` from the short verion) to run install the dev dependencies.

## Tests

In this project, you can run the tests on a descktop or mobile viewport.

### Desktop

Run `npm test`  (or `npm t` from the short verion) to run the test in headless mode on desk top viewport.

Or, run `npm run cy:open` to open the Cypress App on a desktop viewport.

### Mobile

Run `npm run test: mobile` to run the test in headless mode on a mobile viewport.

Or, run `npm run cy:mobile` to open the cypress app on a mobile viewport.
